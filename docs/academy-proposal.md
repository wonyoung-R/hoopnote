## 이렇게 작동합니다

### 1단계: 학생 명단 등록 (1회)

```
이름 / 등번호 / 학부모 연락처
```

- 엑셀 붙여넣기로 한 번에 등록 가능
- 학부모 연락처 = 카카오 알림 자동 발송 대상

### 2단계: 경기 기록 (경기 중)

스마트폰 하나로 체육관에서 한 손으로 조작합니다.

```
┌─────────────────────────────┐
│                              │
│   8개 버튼 — 터치만 하세요    │
│                              │
│  [2점 성공]  [2점 미스]      │
│  [3점 성공]  [3점 미스]      │
│  [자유투 성공] [자유투 미스]  │
│  [리바운드]  [스틸]          │
│                              │
│   → 선수 이름 탭 → 끝!      │
│                              │
└─────────────────────────────┘
```

- **한 동작 3초 이내**
- 엄지 하나로 조작 (체육관 환경 최적화)
- 다크 모드 기본 (눈부심 방지)
- 실수? → 뒤로가기(Undo)로 즉시 수정

### 3단계: 경기 종료 → 자동 처리

경기 끝나면 코치가 할 일:

1. **코치 코멘트 작성** (선수별 한 줄, 선택사항)
2. **"확정" 버튼 터치**
3. **끝.**

HoopNote가 자동으로:
- ✅ 프로급 **박스스코어** 생성
- ✅ 학부모에게 **카카오 알림** 발송
- ✅ 선수별 **누적 통계** 업데이트
- ✅ **월별 성장 리포트** 반영

### 4단계: 학부모가 보는 화면

학부모는 카카오 알림톡을 받고, 터치 한 번으로 확인합니다.

```
┌──────────────────────────────┐
│  📱 카카오 알림톡              │
│                               │
│  [HoopNote] 김민재 경기 결과   │
│                               │
│  🏀 A팀 vs B팀  52 : 48      │
│  📊 김민재: 12점 5리바 2어시   │
│  💬 코치: "3점 슈팅 폼이 많이  │
│     좋아졌습니다. 드리블 연습   │
│     더 하면 좋겠어요."        │
│                               │
│  [상세 보기] →                │
│                               │
└──────────────────────────────┘
```

**학부모가 가족 단톡방에 공유** → 자연스러운 입소문

## 자주 묻는 질문

**Q. 앱을 설치해야 하나요?**
→ 아닙니다. **웹 브라우저**에서 바로 사용합니다. 홈 화면에 추가하면 앱처럼 사용 가능합니다.

**Q. 체육관에서 인터넷이 안 되면?**
→ 기록은 폰에 자동 저장됩니다. 인터넷 연결되면 자동으로 서버에 동기화됩니다.

**Q. 학부모가 별도로 가입해야 하나요?**
→ 아닙니다. 학부모는 카카오 알림톡만 받으면 됩니다. 더 자세히 보고 싶으면 카카오 로그인 한 번이면 끝입니다.

**Q. 보조 코치도 기록할 수 있나요?**
→ 네. 학원 플랜에 코치 3명까지 포함. 각자 기록하고, 학원장이 통합 관리합니다.

**Q. 세금계산서 발행 되나요?**
→ 네. 사업자등록번호로 세금계산서 발행 가능합니다.

**Q. 기존 데이터를 옮길 수 있나요?**
→ 엑셀 선수 명단 붙여넣기로 한 번에 이전 가능합니다. 과거 경기 기록은 수동 입력 필요합니다.

**Q. 학원비 수납 관리도 되나요?**
→ 현재 개발 중입니다. 곧 출시 예정이며, 사전 등록하시면 가장 먼저 안내드립니다.
