📊

출석 패턴 분석

출석 데이터를 AI가 분석하여 "3주 연속 지각", "결석률 상승" 등 이상 패턴을 자동 감지합니다.

💳

미수금 자동 알림

납부기한이 지나면 AI가 자동으로 학부모에게 재알림을 발송합니다. 코치가 직접 연락할 필요 없습니다.

📝

월간 리포트 초안

학생의 출석, 경기 데이터를 AI가 분석하여 코치 총평 초안을 자동 작성합니다. 수정만 하세요.

💬

학부모 FAQ 자동 응답

"수업 시간 언제예요?" 같은 반복 질문에 AI가 자동으로 답변합니다.

⛹️

선수 강점/약점 분석

경기 스탯을 AI가 분석하여 선수별 강점과 보강 포인트를 자동으로 알려줍니다.

📋

수업 일지 자동 생성

출석 체크와 경기 기록을 하면, 오늘의 수업 일지가 자동으로 작성됩니다.

🔔

상담 후속 리마인더

상담 기록에 후속 조치일을 설정하면, AI가 해당일에 맥락과 함께 자동으로 알려줍니다.

자주 묻는 질문

HoopNote는 어떤 서비스인가요?

유소년 농구 학원 코치를 위한 경기 기록, AI 알림장, 학부모 소통 올인원 플랫폼입니다. 코치가 경기를 기록하면 AI가 학부모 알림장을 자동 생성하고, 카카오 알림톡으로 발송합니다.

학부모가 별도 앱을 설치해야 하나요?

아닙니다. 학부모는 카카오톡으로 알림을 받습니다. 카카오 채널 친구 추가만 하시면 됩니다.

보조 코치도 사용 가능한가요?

네. 원장님이 카카오톡으로 초대 링크를 보내면, 코치가 링크 클릭 → 카카오 로그인 → 원장님 승인 후 바로 사용 가능합니다. Pro 플랜부터 코치 수 무제한입니다.

학원비 수납도 되나요?

네. 코치가 수납 알림톡을 보내면 학부모가 카카오톡에서 결제 링크로 바로 결제합니다. 자동 장부 관리는 물론, AI가 미수금을 감지하여 자동으로 재알림을 발송합니다.

카카오 알림톡은 어떤 종류가 있나요?

대회 안내, 수납 안내, 휴강 안내, 보강 안내, 셔틀버스 출발, 셔틀버스 변경 총 6종의 알림톡 템플릿을 제공합니다. 유형을 선택하고 내용만 입력하면 학부모에게 카카오톡으로 즉시 발송됩니다.

무료 체험 후 자동 결제되나요?

아닙니다. 7일 무료 체험 종료 후 직접 구독을 시작해야 합니다. 자동 결제는 없으니 부담 없이 체험해보세요.

기존 학생 데이터를 옮길 수 있나요?

엑셀 붙여넣기로 학생 명단을 한 번에 이전할 수 있습니다. (준비중)
